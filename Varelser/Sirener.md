# Sirener

Hälften kvinna, hälften fågel — lurar skepp på grund och sliter besättningar i stycken (inte att förväxla med sjöjungfrur, som idkar byteshandel vid Fort Mir). **Alluring Song**: wis save eller charmad (rör sig endast mot sirenen; ny save varje runda; ~300 ft). Inga kända svagheter. Tre dräptes vid [[Siren's Call]] — Estalfus överlevde med nöd och näppe.
