# Felicia Bennet

Överbefälhavare i [[Mirfield]]s vaktstyrka. Mötte partyt utanför [[Yttre Torgets syndikat|Yttre Torget]]-räden; ledde utredningen efter [[Edgar]]s kidnappning (rekonstruktionen av anfallet vid östra porten). Argumenterade med [[Pegasus Express]]-bud utanför vakthuset; närvarade vid [[Maxwellbalen]]. Underlydande: Rose.
